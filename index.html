UI5 Test-Recorder

A Chrome Extension to record and generate UI5 Tests

The UI5 Test-Recorder is intended to use for your daily testing of UI5 Applications.
By recording every of your steps, the UI5 Test-Recorder is able to generate the testcode either for Integration Tests with OPA5 or E2E Tests with UIVeri5 or Testcafé.

Available at the Chrome Webstore

The UI5 Test-Recorder is available through the Chrome Webstore.
If you want to get the latest version, not published at the Chrome Webstore, you can:
  • copy the code from master, or developement branch
  • add an ui5-folder at the top level of the UI5 Test-Recorder project
  • paste the resources-folder of an downloaded openUI5 runtime inside the ui5-folder
  • add the UI5 Test-Recorder as an upacked extension to the chrome browser via the chrome extension page

In the future we will use the selfcontained build (by ui5-tooling) for the UI5 Test-Recorder, making the steps above unnecessary.
Except the first and last one. ;)
